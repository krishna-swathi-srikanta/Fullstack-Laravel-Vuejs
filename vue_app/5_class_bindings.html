Class Bindings

  1. class binding is mechanism to dynamically applying classes to the element
  2. we can use v-bind:class="argument"
  3. Argument can be object or array or string

Example :change color on mouseover and mouseout

There are 3 ways to define class

  1. object format => :class="{'bg-info':!hovered,'bg-primary':hovered}"
  2. array format => :class=[hovered? 'bg-primary':'bg-info']
  3. string format => :class="bg-info" , assign class value as stringdynamically onhover
