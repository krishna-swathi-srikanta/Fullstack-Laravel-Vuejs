Style Bindings

Style Binding is mechanism for dynmically applying css definition element use can use v-bind:style="argument" or use can just use :style="argument" Here argument can be array or object

Example : Progress Bar

         1. Style as object 
        here key can be camelcase like fontSize , or snake case like font-size
    
         <div>  :style="mystyle"> something here <div>       
        mystyle:{
            color:blue,
            padding:'2px',

        }

Example : Customize style options

on changing options dynamically change the styling using simple v-model and assign array argument to the style 

         2. Style as array 
        we can combine  2 or more style like this 

        a:{
            color:white,
        }
        b:{
            padding:'2px',
        }
         <div>  :style="[a,b]"> something here <div>
